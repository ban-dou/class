- AR model
- HMM
- Kalman filter
- Kalman smother
- Liner regression
- Liner classification
- PCA
- LDA
- EM algorithm
- VB algorithm
- Gauss sampling
- Particle filter
- Td learning (SARSA, Q-learning)
